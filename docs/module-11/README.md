# Module 11: Video Comments

This module handles the video commenting system, including posting top-level comments, replying to existing comments, viewing comment threads with nested replies, and displaying author information with timestamps.

## Features Overview

| #    | Feature/Transaction                                      | Actor  |
|------|----------------------------------------------------------|--------|
| 11.1 | User can post a comment on a video                       | User   |
| 11.2 | User can reply to an existing comment                    | User   |
| 11.3 | User can view all comments on a video                    | User   |
| 11.4 | User can view nested replies in a comment thread         | User   |
| 11.5 | System displays comment author information and timestamp | System |

## Feature Groups

### [Features 11.1-11.2: Comment Creation](./feature-11.1-11.2/documentation.md)
Covers posting new comments and replying to existing comments.
- 11.1: Post top-level comment on video
- 11.2: Reply to existing comment

### [Features 11.3-11.5: Comment Viewing](./feature-11.3-11.5/documentation.md)
