# Module 14: Desktop App Management

This module handles desktop application management including automatic updates via electron-updater, developer debugging tools, and app download functionality from the web application.

## Features Overview

| #    | Feature/Transaction                                     | Actor  |
|------|---------------------------------------------------------|--------|
| 14.1 | System checks for desktop app updates                   | System |
| 14.2 | System downloads and installs app updates automatically | System |
| 14.3 | User can open DevTools for debugging                    | User   |
| 14.4 | User can download the desktop app from the web app      | User   |

## Feature Groups

### [Features 14.1-14.2: Auto Update System](./feature-14.1-14.2/documentation.md)
Covers automatic update checking and installation.
- 14.1: Check for updates on app startup
- 14.2: Download and install updates automatically

### [Features 14.3-14.4: Developer Tools and App Download](./feature-14.3-14.4/documentation.md)
Covers debugging tools and app distribution.
- 14.3: Open DevTools for debugging
