# Module 13: Search and Discovery

This module handles search functionality across the application, including comprehensive content search (workspaces, folders, videos), quick search with real-time dropdown suggestions, and user search for workspace invitations.

## Features Overview

| #    | Feature/Transaction                                               | Actor  |
|------|-------------------------------------------------------------------|--------|
| 13.1 | User can search across all content (workspaces, folders, videos)  | User   |
| 13.2 | User can search videos by title or description                    | User   |
| 13.3 | User can perform quick search with real-time dropdown suggestions | User   |
| 13.4 | System limits search results to user-accessible content           | System |
| 13.5 | User can search for other users to invite                         | User   |

## Feature Groups

### [Features 13.1-13.4: Content Search](./feature-13.1-13.4/documentation.md)
Covers searching across all content types with access control.
- 13.1: Search workspaces, folders, and videos
- 13.2: Search videos by title or description
- 13.3: Quick search with real-time dropdown and fuzzy matching
- 13.4: Access control - only user-accessible content

### [Feature 13.5: User Search](./feature-13.5/documentation.md)
Covers searching for users to invite to workspaces.
- 13.5: Search users by name or email for invitations
