# Module 12: Notifications

This module handles the notification system including viewing notifications, unread counts, marking as read, and system-triggered notifications for invitations, video views, likes, and uploads.

## Features Overview

| #    | Feature/Transaction                                                      | Actor  |
|------|--------------------------------------------------------------------------|--------|
| 12.1 | User can view all their notifications                                    | User   |
| 12.2 | User can view unread notification count                                  | User   |
| 12.3 | User can mark all notifications as read                                  | User   |
| 12.4 | System creates invitation notification when user is invited to workspace | System |
| 12.5 | System creates video view notification for video owner                   | System |
| 12.6 | System creates video like notification for video owner                   | System |
| 12.7 | System creates video upload notification for workspace members           | System |

## Feature Groups

### [Features 12.1-12.3: Notification Viewing & Management](./feature-12.1-12.3/documentation.md)
Covers viewing and managing notifications.
- 12.1: View all notifications in dropdown
- 12.2: View unread count badge
- 12.3: Mark all as read (auto on dropdown open)

### [Features 12.4-12.7: Notification Creation](./feature-12.4-12.7/documentation.md)
Covers system-triggered notification creation.
- 12.4: Workspace invitation notifications
- 12.5: Video view notifications
- 12.6: Video like notifications
- 12.7: Video upload notifications
